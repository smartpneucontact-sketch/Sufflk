**RFI-0163**  |  Project: Allerton Tower  |  Submitted: 2026-04-08  |  Closed: 2026-04-10
**Discipline:** Structural Steel  |  **Trade:** Steel Erector
**References:** Spec 05 12 00 §3.07

**Question:**
Skidmore-Wilhelm calibration tool used by Erector for slip-critical bolt installation last calibrated 2025-08-12 per attached certificate. Spec implies daily calibration. Confirm tool re-calibration cadence acceptable for current high-rise erection sequence.

**Response (CM QA + EOR):**
Spec 05 12 00 §3.07 calls for daily calibration confirmation, not annual factory re-calibration. Erector shall perform daily verification on installed tool with reference bolt assembly per RCSC Specification §8.2. Submit verification log to CM each shift. Failure to perform daily verification will require re-tensioning of any bolts installed since last verified day.

**Resolution:** Erector implemented daily verification log starting 4/10.
**Schedule impact:** none  |  **Cost impact:** ~$200/shift labor  |  **Escalated to EOR:** yes
