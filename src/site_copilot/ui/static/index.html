What this is

Two AI agents running on the same stack vocabulary Suffolk recruits on, against a synthetic construction corpus (Allerton Tower — a fictional 32-story Boston project named after Suffolk's HQ address).

What's in the agent loop

  • retrieve — hybrid lexical search over project corpus, returns chunks with citations.
  • classify_urgency — low / medium / high based on safety + critical-path + days-until-affected-activity.
  • estimate_impact — rough order-of-magnitude cost and schedule deltas.

What this is not

A one-week portfolio demo, not a production deploy. The corpus is synthetic but realistic. The Terraform deploys infra but no autoscaling or alarms. Procore / OpenSpace integrations are the natural next two — see the README for the honest gaps list.
